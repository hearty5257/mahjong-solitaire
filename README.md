# 心式麻將 · Mahjong Solitaire

可玩的單人麻將解謎遊戲 MVP。React + TypeScript + Vite，純前端，無後端、無登入、無外部素材。

## 啟動

```bash
npm install
npm run dev          # 開發伺服器 (預設 http://localhost:5173)
npm run build        # tsc 型別檢查 + 生產建置 → dist/
npm run preview      # 預覽建置結果
npm test             # 執行 Vitest 測試
```

## 規則

- 點選兩張**相同牌面**且**可移動**的牌即可消除。
- **可移動**＝上方沒有牌覆蓋，且左或右至少一邊開放。
- 全部牌消除 → 勝利。
- 沒有任何可配對的可移動牌 → 卡關。

## 牌組

MVP 採用傳統麻將 136 張牌：
- 筒、條、萬各 1–9（每種 4 張，共 108 張）
- 風牌：東南西北（每種 4 張，共 16 張）
- 三元牌：中發白（每種 4 張，共 12 張）
- 為簡化 MVP，**不包含花牌與季節牌**

## 牌型（v0.2：5 種 + 隨機鏡像）

| Key | 名稱 | 牌位數 | 玩法特色 |
| --- | --- | --- | --- |
| `turtle`     | 經典龜形 | 136 | 中央高、四周低，從外圍打開中央 |
| `pyramid`    | 金字塔   | 120 | 中央最高的階梯方塔，逐層拆解中心 |
| `twinTowers` | 雙塔     | 120 | 兩座 4 層塔 + 中央連接橋，需選擇路線 |
| `bridge`     | 橋樑     | 114 | 左右兩塊平地 + 中央拱橋，先拆橋或先拆兩邊 |
| `spiral`     | 螺旋     | 100 | 同心環狀向中央盤旋，順序影響開放路徑 |

- 每次新局根據 seed 隨機挑一種牌型
- 每局 50% 機率水平鏡像（同 seed 一致）
- 牌型名稱顯示在狀態列

## 魔法道具（v0.3：7 種）

| 道具 | 預設次數 | 稀有度 | 效果 | 扣分 |
| --- | --- | --- | --- | --- |
| 👁 靈視提示 | 3 | Common | 高亮一組可配對的牌，**不消除** | -10 |
| 🔮 預言術 | 1 | Rare | **一次顯示多組可配對牌**（最多 3 對），不消除 | -25 |
| 🌫 透視術 | 1 | Rare | **永久翻開所有迷霧牌**（僅 fog 局有效） | -15 |
| 🗝 解封術 | 1 | Rare | **解除一張**冰封或鎖鏈牌 | -30 |
| ✨ 魔法消除 | 2 | Epic | **自動消除一組**合法的可配對牌（一次只消一組） | -50 |
| ⏪ 時光回溯 | 3 | Common | 回復上一個操作 — **整盤狀態還原**（含 modifier） | -20 |
| 🔀 命運重排 | 1 | Epic | **不改變位置與數量**，僅重新分配剩餘牌的牌面 | -100 |

所有道具每次只執行「一小步」，不會直接幫玩家完成整局。

道具使用優先順序：**先消耗本局基本次數，用完再消耗道具庫**。

## 特殊規則 Round Modifiers（v0.3 新增）

每局新局時，依 seed + 難度可能隨機附加一條特殊規則：

| Round Modifier | 觸發 | 效果 |
| --- | --- | --- |
| **無** | easy 永遠；normal 50% / hard 25% | 不附加任何規則 |
| **迷霧 Fog** | normal / hard 機率出現 | 約 30% 的牌一開始顯示「？」牌背；變成可移動時自動翻開；可用「透視術」永久翻開 |
| **冰封 Frozen** | normal / hard 機率出現 | 約 12% 的牌被冰封不可配對；每消除 3 組自動解凍 1 張；可用「解封術」 |
| **鑰匙與鎖鏈 KeyLock** | normal / hard 機率出現 | 2 對「鑰匙」+ 6 張「鎖鏈」；先消除所有鑰匙，再自動解鎖所有鎖鏈；可用「解封術」單獨解 |

限制：
- easy 難度永遠不附加 round modifier
- 迷霧 / 冰封 不會落在「初始可移動」的牌上 → 避免一開局沒有可操作的牌
- 鑰匙必定是初始可移動的牌 → 確保可在合理步數內取得

## 過關獎勵系統（v0.2 新增）

通關後系統會隨機贈送 1 個魔法道具進入「道具庫」，下一局可使用。

### 稀有度與抽中機率（基本）
- **Common 60%**：靈視提示、時光回溯
- **Rare 30%**：預言術、透視術、解封術
- **Epic 10%**：魔法消除、命運重排

### 表現加成（v0.4）
| 條件 | 效果 |
| --- | --- |
| 困難難度 | common −15、rare +7、epic +8 |
| 全程未使用道具 | common −20、rare +15、epic +5 |
| 時間 < 3 分鐘 | epic +5、common −5 |
| 每日挑戰 | 抽到 common 自動升 rare |
| 最高連擊 ≥ 5 | 60% 機率額外贈送 Hint 或 Undo |

### 持有上限
- Hint / Undo：9
- Reveal / Unseal：6
- Foresight：5
- Magic Remove / Shuffle：3

### 滿額補償
若抽中的道具已達持有上限，改贈分數：
- Common 滿：+50 分
- Rare 滿：+100 分
- Epic 滿：+200 分

### 道具庫
- 存於 `localStorage` key `mahjong-solitaire:inventory:v1`
- 跨局累積、重新整理頁面後仍存在
- 使用道具會正確扣除（不會出現負數）
- 道具按鈕上會顯示 `x 總次數 +N`，其中 N 是道具庫提供的額外次數

## 分數

- 配對成功：+100
- 連擊（4 秒內連續配對）：每連擊額外 +25 × 連擊數
- 道具使用：依上表扣分（分數最低為 0）
- 勝利時根據剩餘時間或道具次數可進一步擴充加成（MVP 已預留空間）

## 已實作功能

1. 完整 Mahjong Solitaire 規則：覆蓋、左右開放、可移動、配對、勝負、卡關
2. **5 種牌型**：經典龜形、金字塔、雙塔、橋樑、螺旋
3. **新局自動隨機挑牌型** + 50% 機率水平鏡像（依 seed 決定）
4. **傳統麻將牌面 SVG**：筒子同心圓點、條子竹節、一條紅雀鳥、萬子宋體、風牌、三元牌
5. seedable 隨機牌局產生（同 seed + 難度 ↔ 同牌局）
6. 難度：簡單 / 普通 / 困難（依初始可配對數要求）
7. **7 種魔法道具**：靈視提示、預言術、透視術、解封術、魔法消除、時光回溯、命運重排
8. **過關獎勵 + 道具庫**：隨機抽道具、稀有度權重、持有上限、滿額分數補償
9. **localStorage 持久化**：最佳分數 + 跨局道具庫
10. 分數、步數、即時計時、剩餘牌數、即時可配對數
11. 連擊 Combo
12. 勝利 overlay 顯示獎勵詳細、卡關 overlay 可用 Undo / Shuffle 救援
13. RWD：手機螢幕下排版自動調整
14. **特殊規則 Round Modifiers**：無 / 迷霧 / 冰封 / 鑰匙與鎖鏈，依難度自動套用
15. **Tile Modifiers**：fog / frozen / locked / key，視覺有獨立 overlay 與顏色
16. **板面快照 Undo**：能完整還原 modifier 狀態
17. **每日挑戰**：固定日期 seed，今日只結算一次；困難難度 + 保底 Rare 道具
18. **通關表現加成**：未用道具 / 快速通關 / 困難 / 連擊 → 提高稀有度權重 + 額外贈品
19. **獎勵動畫**：彈出 + 金光 shimmer + 加成 chip 列表
20. 純函式核心邏輯 + Vitest **77** 個測試

## 已知限制與後續可改進

- **不保證 100% 可解**：產生器只保證開局至少 N 組可用配對（依難度），未進行真正的「可解牌局」生成（這需要先用回推法把空盤反向填牌）。在 MVP 範圍內若遇到中途卡關，可使用「時光回溯」或「命運重排」救援，或開始新局。
- **不含花牌與季節牌**：傳統 8 張花季牌的「同類即可配對」機制 MVP 暫未實作。
- **音效未加入**：可用 Web Audio API 進一步擴充。
- **新手教學 / 每日挑戰**：已預留架構（seed + difficulty 已抽象），可在後續加上日期種子與教學 overlay。
